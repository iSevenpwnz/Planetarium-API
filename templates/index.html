Planetarium API

Ласкаво просимо до API Планетарію! Нижче наведено посилання на основні ресурси та інструменти.

Основні ресурси API (префікс /api/):

  • Users (/api/users/, /api/users/me/, /api/users/register/)
  • Tokens (/api/token/, /api/token/refresh/)
  • Astronomy Shows (/api/astronomy-shows/)
  • Show Themes (/api/show-themes/)
  • Planetarium Domes (/api/planetarium-domes/)
  • Show Sessions (/api/show-sessions/, /api/show-sessions/{id}/book/, /api/show-sessions/{id}/available-seats/)
  • Reservations (/api/reservations/)
  • Tickets (/api/tickets/)

Інші посилання:

  • API Root (стандартний вигляд DRF)
  • Swagger UI (детальна документація API)
  • ReDoc (альтернативна документація API)
  • Django Admin (адміністративний інтерфейс)
